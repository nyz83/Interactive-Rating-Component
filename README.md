Interactive Rating Component
  This project is an interactive rating component that allows users to select a rating from 1 to 5 stars. The component displays the current rating and the number of total ratings. The user can hover over each star to see a tooltip that shows the rating value.

Technologies Used
  HTML
  CSS
  JavaScript
  
Usage
  To use the Interactive Rating Component, simply hover over each star to see the tooltip and click on the star to select a rating. The current rating and number of total ratings will be updated in real-time.

Additional Features
In addition to the basic functionality, the component includes the following features:

Smooth animations for hover effects and rating updates.
